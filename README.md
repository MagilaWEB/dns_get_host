# DNS Get Host

**DNS Get Host** — это высокопроизводительная утилита для Windows, написанная на современном **C++23**, которая автоматически анализирует списки доменов и извлекает из них API-маршруты (DNS-хосты).

## 🚀 Быстрый старт

Для работы программы выполните следующие шаги:
1.  **Подготовьте данные**: Создайте папку **`domains`** в вашей пользовательской директории (`user\domains`).
2.  **Добавьте списки доменов**: Поместите в папку `domains` один или несколько файлов с расширением **`.list`**. Каждая строка в файле должна содержать один домен (например, `example.com` или `api.service.io`).
3.  **Запустите программу**: Запустите скомпилированный исполняемый файл.

**Результат работы**: После обработки всех доменов из `.list` файлов, программа сохранит полученные API-маршруты (DNS-хосты) в новую папку **`user\hosts`**. Каждому исходному `.list` файлу будет соответствовать свой файл с результатами в папке `hosts`.

## 🛠 Технические детали

*   **Платформа:** Windows 10/11
*   **Язык и стандарт:** **C++23**
*   **Система сборки:** CMake

## 📁 Структура проекта и сборка

Основной исходный код находится в директории `src`. Для компиляции проекта используйте предоставленные скрипты:
*   `CMakeLists.txt` — основной конфигурационный файл для CMake.
*   `configure.bat` / `configure-msvc.bat` — скрипты для предварительной настройки среды сборки.
*   `build.bat` — скрипт для запуска процесса сборки.
